⚡ PortFree

一款用於快速檢查和釋放 macOS 上被佔用連接埠的開發者工具。

快速開始

在輸入欄中輸入連接埠號（1–65535），點擊 檢查 查看佔用該連接埠的處理程序。如果發現處理程序，可以點擊 結束處理程序 或 強制結束 來終止它。

主要功能

  • 連接埠檢查 — 查看任何連接埠的處理程序名稱、PID、使用者、協定和端點資訊。
  • 快捷連接埠 — 一鍵存取常用連接埠（80、443、3000、5173、8080），還可以新增自訂連接埠。
  • 掃描所有連接埠 — 查看系統上所有正在監聽的 TCP 連接埠。
  • 選單列 — 從選單列快速檢查連接埠，無需開啟主視窗。
  • 歷史記錄 — 保存最近的檢查和操作記錄，方便快速查閱。

全域快速鍵

在任何地方按下 ⌘ ⇧ P 即可將 PortFree 視窗切換到前台。

命令列工具 (fp)

在側邊欄的設定部分安裝命令列工具。安裝後，在終端機中使用：

  • fp 3000 — 查看佔用 3000 連接埠的處理程序，並可選擇終止。
  • fp list — 列出所有監聽中的 TCP 連接埠。
  • fp help — 顯示命令列輔助說明。

管理員權限

某些系統處理程序需要管理員權限才能終止。PortFree 會在需要時自動提示輸入密碼。

設定

  • 開機啟動 — 登入時自動啟動 PortFree。
  • 語言 — 可從 7 種語言中選擇，或跟隨系統設定。
  • 安裝 CLI — 安裝或解除安裝 fp 命令列工具。
