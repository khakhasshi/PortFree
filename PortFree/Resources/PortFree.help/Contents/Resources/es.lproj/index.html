⚡ PortFree

Una herramienta para desarrolladores para verificar y liberar rápidamente puertos ocupados en macOS.

Primeros pasos

Ingresa un número de puerto (1–65535) en el campo de entrada y haz clic en Verificar para ver qué proceso está usando ese puerto. Si se encuentra un proceso, puedes terminarlo con Terminar proceso o Forzar cierre.

Funciones principales

  • Verificación de puertos — Ver el nombre del proceso, PID, usuario, protocolo y punto final de cualquier puerto.
  • Puertos rápidos — Acceso con un clic a puertos comunes (80, 443, 3000, 5173, 8080). Agrega tus propios puertos personalizados.
  • Escanear todos los puertos — Ver todos los puertos TCP en escucha en tu sistema.
  • Barra de menús — Accede a PortFree desde la barra de menús para verificaciones rápidas sin abrir la ventana principal.
  • Historial — Las verificaciones y acciones recientes se guardan para referencia rápida.

Atajo global

Presiona ⌘ ⇧ P desde cualquier lugar para traer la ventana de PortFree al frente.

Herramienta CLI (fp)

Instala el CLI desde la sección de Ajustes en la barra lateral. Una vez instalado, úsalo en la Terminal:

  • fp 3000 — Verificar qué proceso está usando el puerto 3000 y terminarlo si es necesario.
  • fp list — Listar todos los puertos TCP en escucha.
  • fp help — Mostrar la ayuda del CLI.

Privilegios de administrador

Algunos procesos del sistema requieren privilegios de administrador para ser terminados. PortFree solicitará automáticamente tu contraseña cuando sea necesario.

Ajustes

  • Iniciar al arrancar — Iniciar PortFree automáticamente al iniciar sesión.
  • Idioma — Elegir entre 7 idiomas o seguir la configuración del sistema.
  • Instalar CLI — Instalar o desinstalar la herramienta de línea de comandos fp.
