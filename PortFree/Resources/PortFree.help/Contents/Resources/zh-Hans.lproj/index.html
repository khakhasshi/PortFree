⚡ PortFree

一款用于快速检查和释放 macOS 上被占用端口的开发者工具。

快速开始

在输入框中输入端口号（1–65535），点击 检查 查看占用该端口的进程。如果发现进程，可以点击 结束进程 或 强制结束 来终止它。

主要功能

  • 端口检查 — 查看任何端口的进程名称、PID、用户、协议和端点信息。
  • 快捷端口 — 一键访问常用端口（80、443、3000、5173、8080），还可以添加自定义端口。
  • 扫描所有端口 — 查看系统上所有正在监听的 TCP 端口。
  • 菜单栏 — 从菜单栏快速检查端口，无需打开主窗口。
  • 历史记录 — 保存最近的检查和操作记录，方便快速查阅。

全局快捷键

在任何地方按下 ⌘ ⇧ P 即可将 PortFree 窗口切换到前台。

命令行工具 (fp)

在侧栏的设置部分安装命令行工具。安装后，在终端中使用：

  • fp 3000 — 查看占用 3000 端口的进程，并可选择终止。
  • fp list — 列出所有监听中的 TCP 端口。
  • fp help — 显示命令行帮助。

管理员权限

某些系统进程需要管理员权限才能终止。PortFree 会在需要时自动提示输入密码。

设置

  • 开机启动 — 登录时自动启动 PortFree。
  • 语言 — 可从 7 种语言中选择，或跟随系统设置。
  • 安装 CLI — 安装或卸载 fp 命令行工具。
