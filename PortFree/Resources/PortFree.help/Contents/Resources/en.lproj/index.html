⚡ PortFree

A developer tool for quickly inspecting and freeing occupied ports on macOS.

Getting Started

Enter a port number (1–65535) in the input field and click Inspect to see which process is using that port. If a process is found, you can terminate it with End Process or Force End.

Main Features

  • Port Inspection — Check any port to see the process name, PID, user, protocol, and endpoint.
  • Quick Ports — One-click access to common ports (80, 443, 3000, 5173, 8080). Add your own custom ports.
  • Scan All Ports — View all currently listening TCP ports on your system.
  • Menu Bar — Access PortFree from the menu bar for quick port checks without opening the main window.
  • History — Recent inspections and actions are saved for quick reference.

Global Hotkey

Press ⌘ ⇧ P from anywhere to bring the PortFree window to the front.

CLI Tool (fp)

Install the CLI from the Settings section in the sidebar. Once installed, use it in Terminal:

  • fp 3000 — Check which process is using port 3000 and optionally kill it.
  • fp list — List all listening TCP ports.
  • fp help — Show CLI help.

Admin Privileges

Some system processes require administrator privileges to terminate. PortFree will automatically prompt for your password when needed.

Settings

  • Launch at Login — Start PortFree automatically when you log in.
  • Language — Choose from 7 languages or follow the system setting.
  • Install CLI — Install or uninstall the fp command-line tool.
