⚡ PortFree

macOS 上で使用中のポートを素早く確認・解放する開発者向けツールです。

はじめに

入力欄にポート番号（1〜65535）を入力し、検査 をクリックして、そのポートを使用しているプロセスを確認します。プロセスが見つかった場合は、プロセスを終了 または 強制終了 で終了できます。

主な機能

  • ポート検査 — 任意のポートのプロセス名、PID、ユーザー、プロトコル、エンドポイント情報を確認。
  • クイックポート — よく使うポート（80、443、3000、5173、8080）にワンクリックでアクセス。カスタムポートの追加も可能。
  • 全ポートスキャン — システム上のすべてのリスニング中の TCP ポートを一覧表示。
  • メニューバー — メニューバーからメインウィンドウを開かずにポートをすばやく確認。
  • 履歴 — 最近の検査と操作が保存され、すぐに参照可能。

グローバルホットキー

どこからでも ⌘ ⇧ P を押すと、PortFree ウィンドウが前面に表示されます。

CLIツール (fp)

サイドバーの設定セクションから CLI をインストールします。インストール後、ターミナルで使用できます：

  • fp 3000 — ポート 3000 を使用しているプロセスを確認し、必要に応じて終了。
  • fp list — すべてのリスニング中の TCP ポートを一覧表示。
  • fp help — CLI ヘルプを表示。

管理者権限

一部のシステムプロセスは終了するために管理者権限が必要です。PortFree は必要に応じて自動的にパスワード入力を求めます。

設定

  • ログイン時に起動 — ログイン時に PortFree を自動起動。
  • 言語 — 7 つの言語から選択するか、システム設定に従います。
  • CLI インストール — fp コマンドラインツールのインストール・アンインストール。
