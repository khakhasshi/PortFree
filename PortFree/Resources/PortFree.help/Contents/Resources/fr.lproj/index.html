⚡ PortFree

Un outil de développement pour vérifier et libérer rapidement les ports occupés sur macOS.

Pour commencer

Saisissez un numéro de port (1–65535) dans le champ de saisie et cliquez sur Vérifier pour voir quel processus utilise ce port. Si un processus est trouvé, vous pouvez le terminer avec Terminer le processus ou Forcer la fermeture.

Fonctionnalités principales

  • Vérification de port — Voir le nom du processus, le PID, l'utilisateur, le protocole et le point de terminaison de n'importe quel port.
  • Ports rapides — Accès en un clic aux ports courants (80, 443, 3000, 5173, 8080). Ajoutez vos propres ports personnalisés.
  • Scanner tous les ports — Afficher tous les ports TCP en écoute sur votre système.
  • Barre de menus — Accédez à PortFree depuis la barre de menus pour des vérifications rapides sans ouvrir la fenêtre principale.
  • Historique — Les vérifications et actions récentes sont enregistrées pour une référence rapide.

Raccourci global

Appuyez sur ⌘ ⇧ P de n'importe où pour mettre la fenêtre PortFree au premier plan.

Outil CLI (fp)

Installez le CLI depuis la section Réglages dans la barre latérale. Une fois installé, utilisez-le dans le Terminal :

  • fp 3000 — Vérifier quel processus utilise le port 3000 et le terminer si nécessaire.
  • fp list — Lister tous les ports TCP en écoute.
  • fp help — Afficher l'aide du CLI.

Privilèges administrateur

Certains processus système nécessitent des privilèges administrateur pour être terminés. PortFree vous demandera automatiquement votre mot de passe si nécessaire.

Réglages

  • Lancer au démarrage — Démarrer PortFree automatiquement à la connexion.
  • Langue — Choisir parmi 7 langues ou suivre les réglages système.
  • Installer CLI — Installer ou désinstaller l'outil en ligne de commande fp.
